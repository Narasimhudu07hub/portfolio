1. Spam Mail Prediction Using Machine Learning

Title of the project is "Spam Mail Prediction Using Machine Learning" and it was a group projet by three members. It was my main project during my academics i.e Bachelore of Technology. In this project we build a classifier with builded model by using this classifier we will classify the received email is a "Spam mail" or "Ham mail" Let see what is the working of project which is mentioned below..

Working

  • Let see how the project is working and what are the stages in this..
Aim:

The main goal of the project is build a model to predict and classify the emails i.e either email is spam mail or ham mail.


Modules:

There are two modules in this project. They are System Module & User Module.


Analysis:

With analysis, get to know and differentiate the spam mails and ham mails (not spam) using visualization techniques.(Pie-chart, Bar-Graph,Histogram..etc)


Model Built:

Build a model using Na√Øve Bayes(supervised machine learning algorithm) which is used for classification tasks, like text classification which works on conditional probability and for converting text to numerical TfidfVectorizer is used.


Result:

Get output as either 0 (spam) or 1 (not spam) with accuracy score 98.24% through web page using streamlit open-source framework.


Usecase:

The model outcome could help to email users(business) to identify spam mails
